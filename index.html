Apa itu Framework We2py

Apa itu Framework

Sebelum masuk kedalam pemahaman tentang apa itu Framework Web2py. Pertama-tama perlu di ketahui apakah yang dimaksud dengan Framework itu sendiri.
Framework adalah kerangka-kerja yang meruapakan sekumpulan fungsi, class dan aturan-aturan. Berbeda dengan library yang sifatnya untuk tujuan tertentu saja, framework bersifat menyeluruh mengatur bagaimana kita membangun aplikasi.

Jadi dapat disimpulkan bahwa Web2py adalah framework aplikasi web berbasis open source yang digunakan untuk pengembangan secara cepat, berskala, aman dan untuk aplikasi database-driven berbasis web yang portable. Framework web2py di tulis serta di jalankan menggunakan bahasa pemrograman Python dan dapat dijalankan secara langsung.

Sedangkan bahasa pemrograman yang digunakan pada framework Web2py adalah bahasa pemrograman Python.
Python dapat diartikan sebagai bahasa pemrograman tingkat tinggi yang interpreted dan berorientasi objek. Merupakan bahasa pemrograman yang dibangun dengan struktur data tingkat tinggi, menggabungkan antara dynamic typing dan dynamic binding yang membuatnya menjadi semakin baik untuk pengembangan aplikasi secara cepat.

Database Abstraction Layer

Database Abstraction Layer

Database Abstraction Layer (DAL) merupakan salah satu hal utama dari framework web2py. Database Abstraction Layer menampilan Aplications Programming Interface (API) sederhana untuk membuat sintaks SQL, dan inilah yang menjadi hal utama dari web2py.

Aplications Programming Interface (API) dapat digunakan untuk menjalankan objek Python ke dalam objek database seperti queries, tables dan records. DAL secara dinamis menghasilkan SQL secara real time sehingga pengguna tidak perlu menuliskan kode SQL lagi, ini membuat aplikasi menjadi poratabel untuk berbagai jenis database.

Secara default Web2py menggunakan database SQLite untuk penggunaan Model-nya. SQLite sendiri adalah database yang berukuran kecil dan tidak seperti database client-server yang lain, SQLite tidak berdiri sendiri untuk proses komunikasi pada programnya, melainkan menggunakan library dari SQLite yang menjadi bagian dari program aplikasi tersebut.
